# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-24

### Added
- Initial release of dv-tui
- Multi-tab browsing for JSON and CSV files
- Vim keybindings (j/k, h/l) for navigation
- Fuzzy search with smart character-distance scoring
- Auto-reload for modified files
- Color-coded display for statuses and types
- Support for nested data (drill-down navigation)
- Enum cell editing with predefined values
- Trigger system for executing commands on events
- Custom keybindings via CLI or config file
- Library API for programmatic usage
- Configuration system with file, inline JSON, and CLI support
- Selection modes (row and cell)
- Refresh system for auto-updating data from commands
